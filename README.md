# 寒假学习打卡（git 练手项目）说明

1. fork 本项目到自己的账户
1. 按照打卡格式使用 git 每天更新打卡记录
1. 寒假快结束时使用 pull request 功能合并到本项目

打卡格式：每位同学新起一段二级标题，记录学习日期，也可附上任何其他内容:)

Tips:

1. 本文件用 markdown 语言编写，了解 markdown 语言及基本使用方式可以参考下面列出的链接。 
1. 寒假结束后，打卡天数超过10天的同学可获得小奖励
